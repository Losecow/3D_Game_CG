# 3D 수박게임 개발 계획

> WebGL(Three.js) 기반 개인 게임 개발 프로젝트  
> Suika Game (수박게임) 3D 버전 구현

---

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 렌더링 | Three.js (WebGL) |
| 물리 엔진 | cannon-es |
| 빌드 도구 | Vite |
| 언어 | JavaScript (ES Modules) |
| 주석 | 한국어 + 영어 이중 주석 |

---

## 게임 규칙

1. 컨테이너(박스형) 위에서 과일을 마우스로 조준 후 클릭해 떨어뜨린다
2. 같은 종류의 과일이 닿으면 합쳐져 한 단계 위의 과일이 된다
3. 수박(최고 단계) 두 개가 합쳐지면 사라지며 최고 점수를 획득
4. 과일이 컨테이너 상단 위험선을 넘으면 게임 오버
5. 다음에 떨어질 과일은 미리 확인할 수 있다

## 폴더 구조

```
suika-3d/
├── index.html          # 게임 진입점
├── style.css           # 전역 스타일 + UI 오버레이 스타일
├── main.js             # 엔트리 포인트, Game 인스턴스 생성
├── PLAN.md             # 이 파일
└── src/
    ├── Game.js         # 메인 게임 루프 & 상태 관리
    ├── World.js        # cannon-es 물리 세계 설정
    ├── Container.js    # 컨테이너 (바닥 + 4면 벽)
    ├── Fruit.js        # 과일 클래스 (Three.js mesh + cannon body 결합)
    ├── FruitData.js    # 11단계 과일 상수 데이터
    ├── Merger.js       # 합체 감지 & 처리 로직
    └── UI.js           # HTML 오버레이 (점수, 다음 과일, 게임오버)
```

## 개발 단계 (Milestones)

### Phase 1 — 기본 씬 구성
- [x] cannon-es 설치 및 Three.js 연동
- [x] 게임 루프 (requestAnimationFrame + 물리 스텝)
- [x] 박스형 컨테이너 생성 (바닥 + 좌/우/앞/뒤 4면 벽, 투명)
- [x] OrbitControls 카메라 (마우스 드래그로 360도 회전)
- [x] 위험선 시각화 (컨테이너 상단 빨간 라인)

### Phase 2 — 과일 드롭 시스템
- [x] FruitData 정의
- [x] Fruit 클래스 (Three.js SphereGeometry + cannon Sphere Body)
- [x] 마우스 클릭으로 컨테이너 위에서 과일 낙하
- [x] 과일 가이드 라인 (드롭 위치 미리보기)
- [x] 다음 과일 미리보기 로직

### Phase 3 — 합체 시스템
- [x] cannon-es 충돌 이벤트로 접촉 감지
- [x] 같은 단계 과일 합체 → 중간 위치에 다음 단계 과일 스폰
- [x] 합체 시 이펙트 (간단한 스케일 애니메이션)
- [x] 수박끼리 합체 시 소멸 처리

### Phase 4 — 게임 오버 & 점수
- [x] 컨테이너 상단 초과 감지 (일정 시간 이상 위험선 위에 있을 때)
- [x] 점수 집계 (합체 이벤트마다 누적)
- [x] 최고 점수 localStorage 저장
- [x] 게임 오버 화면 + 재시작 버튼

### Phase 5 — 완성도 향상
- [x] 과일 드롭 시 쿨다운 (연속 클릭 방지)
- [x] 합체 사운드 이펙트 (Web Audio API)
- [x] 반응형 레이아웃
- [x] 성능 최적화 (지오메트리 캐싱, 스프라이트 dispose)
